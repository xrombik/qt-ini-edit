# qt-ini-edit
QT training ground

Сборка в Windows:
```build.bat```

Сборка в Linux:
```build.sh```

Запуск в Windows:
```qt-ini-edit.bat```

Запуск в Linux:
```qt-ini-edit.sh```

Запуск тестов в Windows:
```run-tests.bat```

Запуск тестов в Linux:
```run-tests.sh```
